# Git HOL 4 - Conflict Resolution

## Objective
This hands-on exercise demonstrates how merge conflicts occur and how to resolve them.

## Branch Used
- GitWork

## Conflict File
- hello.xml

## What I Understood
A merge conflict happens when Git cannot automatically combine changes from two branches. In this exercise, the same file was created differently in main and GitWork. Git showed conflict markers, and the file had to be manually corrected before committing the resolved version.
