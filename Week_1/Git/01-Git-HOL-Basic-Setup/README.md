# Git HOL 1 - Basic Git Configuration and Repository Setup

## Objective
This hands-on exercise covers basic Git setup, creating a local repository, adding a file, checking Git status, committing changes, and preparing to push to a remote repository.

## Commands Practised
- git config --global
- git init
- dir -Force
- git status
- git add
- git commit
- git remote
- git push

## File Created
- welcome.txt

## What I Understood
Git tracks changes inside a working directory. A new file first appears as untracked. After using git add, the file moves to the staging area. After git commit, the change is saved in the local repository history. A remote repository like GitHub is used to store and share the committed work online.
