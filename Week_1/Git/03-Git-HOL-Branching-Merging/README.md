# Git HOL 3 - Branching and Merging

## Objective
This hands-on exercise demonstrates Git branching and merging.

## Branch Used
- GitNewBranch

## Commands Practised
- git branch
- git switch
- git status
- git add
- git commit
- git diff
- git merge
- git log --oneline --graph --decorate
- git branch -d

## What I Understood
A branch allows work to happen separately from the main branch. After changes are completed and committed in the feature branch, they can be merged back into main. After a successful merge, the feature branch can be deleted because the work is already included in main.
