# Git HOL 5 - Cleanup and Push to Remote

## Objective
This hands-on exercise demonstrates cleanup, pulling from remote, pushing local changes, and verifying that the remote repository is updated.

## Commands Practised
- git status
- git branch -a
- git pull origin main
- git push
- git log --oneline --graph --decorate

## What I Understood
Before pushing work to a remote repository, the local branch should be clean and up to date. The git pull command brings the latest remote changes into the local branch, and git push uploads local commits to the remote repository. Branch cleanup keeps the repository organised after merge work is completed.
