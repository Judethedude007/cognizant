# Exercise 1 - PL/SQL Control Structures

## Objective
Implement PL/SQL control structures using loops and IF conditions for banking scenarios.

## Scenarios Completed
1. Applied a 1% discount to loan interest rates for customers above 60 years old.
2. Promoted customers with balance greater than 10000 to VIP status.
3. Printed reminder messages for loans due within the next 30 days.

## Files Created
- schema_setup.sql
- scenario1_loan_discount.sql
- scenario2_vip_status.sql
- scenario3_loan_reminders.sql

## How to Run
Use Oracle SQL Developer or Oracle Live SQL.

Run the scripts in this order:
1. schema_setup.sql
2. scenario1_loan_discount.sql
3. scenario2_vip_status.sql
4. scenario3_loan_reminders.sql

Before running the PL/SQL blocks, enable DBMS output if using SQL Developer.

## What I Understood
PL/SQL control structures allow procedural logic to be used with database operations. In this exercise, FOR loops were used to process multiple records, and IF conditions were used to apply business rules such as senior-citizen discounts, VIP status updates, and loan reminder messages.
